Dune Studio Website.

A portfolio and studio presence for an interior design firm — project galleries, services pages, press coverage, and a new client enquiry flow.

Challenge

Dune Studio's portfolio was a PDF. They were pitching for hospitality and high-end residential projects but had no digital portfolio to send to prospective clients or attach to award submissions.

Strategy

We designed a presentation-first site: a full-screen project gallery that loaded in a sequence ordered by project type (residential, hospitality, commercial). Each project page had a brief, location, scope, and photography carousel. A services page articulated the studio's five offering areas. An enquiry form routed by project type.

Result

Client enquiries via the website generated four qualified briefs in the first quarter. The studio was accepted to a regional design award shortlist citing the website as the submission vehicle. Project page average session duration was 3m 48s.
